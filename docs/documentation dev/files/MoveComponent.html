Detailed Description

Component used for simple position manipulation operations on object. Gives tools for moving and setting position of object.

Usage Example
//Create Engine
yen::Engine engine;
EngineConfiguration configuration;
engine.initialize(configuration);

//Creating object
yen::ObjectManipulator object = engine.objectsManager->createObject(yen::fVector(0.0f, 0.0f));

//Creating and attaching Move Component. Start position is 0.0f, 0.0f.
yen::MoveComponent moveComponent(yen::fVector(0.0f, 0.0f));
yen::ComponentManipulator moveComponentManipulator;
engine.objectsManager->attachComponent(&moveComponentManipulator, object, &moveComponent); 

//Create scene and object to it
yen::SceneManipulator scene = engine.sceneManager->createScene();
engine.sceneManager->addObjectToScene(scene, object);

void move (yen::fVector)

Method moves object arround scene by given value.

For example if object is on position X = 5.0f, Y= 14.5f, after move by X = -1.2f, Y = 0.5f new position of object would be: X = 3.8f, Y= 15.0f.

Arguments:

yen::fVector - move value.
