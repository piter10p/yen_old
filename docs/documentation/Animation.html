Detailed Description

This class loads animations to memory and unloads it, contains all frames and controls animation playing.

Usage Example
        //Creating Engine
yen::Engine engine;
EngineConfiguration configuration;
engine.initialize(configuration);
                
//Creating animation
yen::AnimationResourceDef animationDef;
animationDef.path = "anim";//Folder name
animationDef.frameRate = 60;//Animation speed is 60 frames per second
yen::AnimationManipulator animation = engine.resourceManager->addAnimationResource(animationDef);
                
//Creating Graphics Object
yen::ObjectManipulator cube = engine.objectsManager->createObject(yen::fVector(0.0f, 0.0f));
engine.objectsManager->setLoadDistance(cube, 500.0f);

engine.sceneManager->setSceneGravity(scene, yen::fVector(0.0f, 10.0f));
                
//Adding object to scene
engine.sceneManager->addObjectToScene(scene, cube);
                
//Scene initializing
engine.sceneManager->initializeScene(scene);
Member Methods Documentation
void load (std::string path)

Method loads animation frames from all files in folder (in order by file name) to memory.

Arguments:

std::string path - path to folder from animation have to be loaded.

void unLoad ()

Method unloads animation frames from memory.

void setFrameRate (unsigned int)

Method sets animation frame rate.

Frame rate value is expressed in frames per second.

Arguments:

unsigned int - frame rate.

unsigned int getFrameRate ()

Method returns animation frame rate.

Frame rate value is expressed in frames per second.

Returns:

unsigned int - frame rate.

Frame* getActualFrame ()

Method returns animation's actual frame.

Returns:

Frame* - actual frame.

void start ()

Method makes animation played.

void pause ()

Method makes animation paused.
