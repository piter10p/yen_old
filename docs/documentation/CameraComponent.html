Detailed Description

An abstract class used for creating Component classes.

Usage Example
        //Create Engine
yen::Engine engine;
EngineConfiguration configuration;
engine.initialize(configuration);

//Camera Object declaration
yen::ObjectManipulator camera = engine.objectsManager->createObject(yen::fVector(0.0f, 0.0f));

//CameraComponent declaration and using
yen::CameraComponent cameraComponent(engine.graphicsEngine);
cameraComponent.setSize(yen::fVector(engine.settingsManager->getGraphicsSettings().resolution.getX(), engine.settingsManager->getGraphicsSettings().resolution.getY()));

//Set View Port of CameraComponent. Camera will be rendered in half of width screen. Size of View is setted to half of window width.
cameraComponent.setViewPort(yen::fRect(yen::fVector(0.5f, 0.0f), yen::fVector(0.5f, 1.0f)));

//Attach CameraComponent to Camera object
yen::ComponentManipulator cameraComponentManipulator;
engine.objectsManager->attachComponent(&cameraComponentManipulator, camera, &cameraComponent);

//Create scene and add Camera object to it
yen::SceneManipulator scene = engine.sceneManager->createScene();
engine.sceneManager->addObjectToScene(scene, camera);

engine.sceneManager->initializeScene(scene);

void setSize (yen::fVector)

Method sets size of camera.

Size value means area of scene that camera recording.

Arguments:

yen::fVector - size of camera

void setZoom (float)

Method sets zoom of camera.

Zoom value affects on area of scene that camera recording. Lower value gives smaller area.

Arguments:

float - value of zoom.

void setRotation (float angle)

Method sets rotation of camera.

Arguments:

float - value of rotation in degrees.

void setViewPort (yen::fRect)

Method sets View Port of camera.

View Port value determines how that, what camera see will be rendered in window.

View Port values is an factor.

Arguments:

yen::fRect - View Port.

X and Y values means position on screen (0.0f and 0.0f is top left corner, 0.5f, 0.0f is right side of window).

Width and Height values means size of View (1.0f and 1.0f means size of window, 0.5f, 0.5f means a quarter of a window).

void setActive (bool)

Method sets camera active or unactive.

Active camera render that, what it see on screen.

bool isActive ()

Method returns that, is camera active.

void draw (std::vector <GraphicsComponent*>*)

Method draws objects of scene from camera perspective.

Arguments:

std::vector <GraphicsComponent*>* - list of GraphicsComponents of objects on scene.
