An abstract class for all YEN Components.

Detailed Description

An abstract class used for creating Component classes.

Usage Example
        class CustomComponent: public yen::Component
{
    CustomComponent()
    {
        this->type = "CustomComponent";
    }

    void codeStepUpdate(ObjectAccessInterface i)
    {
        //Some operations here
    }
    void initialization(ObjectAccessInterface i)
    {
        //Some operations here
    };
    void setResourcesUsed(){}
    {
        //Some operations here
    }

    void freezed()
    {
        //Some operations here
    }
    void unFreezed()
    {
        //Some operations here
    }
};

An abstract method called on component initialization. Used for prepare component to work.
