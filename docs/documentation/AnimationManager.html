Detailed Description

This class is used by Graphics Component for manage animations.

Member Methods Documentation

Arguments:

yen::AnimationManipulator - manipulator of animation.

Method adds animation resource to animations used by manager.

Arguments:

yen::AnimationManipulator - manipulator of animation.

Throws:

yen::ManipulatorError

void removeAnimation (yen::AnimationManipulator)

Method removes animation resource from animations used by manager.

Arguments:

yen::AnimationManipulator - manipulator of animation.

Throws:
