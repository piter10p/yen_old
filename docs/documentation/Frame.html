class yen::Frame
Frame of animation. Contains image and loads it.

Detailed Description

This class stores one frame of animation and loads frame from image.

Member Methods Documentation
yen::Flag load (std::string path)

Method loads frame image from file to memory.

Arguments:

std::string path - path to folder from animation have to be loaded.

Returns:

yen::Flag - yen::Flag::ERROR_CAN_NOT_OPEN_FILE if can not load file.

sf::Texture* getTexture ()

Method returns pointer to stored frame image.

Returns:

>sf::Texture* - pointer to image.
